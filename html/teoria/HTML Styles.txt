L'attributo HTML styleviene utilizzato per aggiungere stili a un elemento, come colore, carattere, dimensione e altro ancora.
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
L'attributo di stile HTML
L'impostazione dello stile di un elemento HTML può essere effettuata tramite l' styleattributo .
L' style attributo HTML ha la seguente sintassi:

<tagname style="property:value;">
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Colore di sfondo
La background-colorproprietà CSS definisce il colore di sfondo per un elemento HTML.

Imposta il colore di sfondo di una pagina su azzurro polvere:

<body style="background-color:powderblue;">
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
</body>


Imposta il colore di sfondo per due elementi diversi:

<body>
<h1 style="background-color:powderblue;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>
</body>
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Colore del testo
La colorproprietà CSS definisce il colore del testo per un elemento HTML:

Esempio
<h1 style="color:blue;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Caratteri
La proprietà CSS font-familydefinisce il font da utilizzare per un elemento HTML:

Esempio
<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Caratteri
La proprietà CSS font-familydefinisce il font da utilizzare per un elemento HTML:

Esempio
<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Dimensione del testo
La font-sizeproprietà CSS definisce la dimensione del testo per un elemento HTML:

Esempio
<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Utilizzare l' styleattributo per definire lo stile degli elementi HTML
Utilizzare background-colorper il colore di sfondo
Utilizzare colorper i colori del testo
Utilizzare font-familyper i caratteri di testo
Utilizzare font-sizeper le dimensioni del testo
Utilizzare text-alignper l'allineamento del testo
